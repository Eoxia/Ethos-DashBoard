Ethos-DashBoard allow a simple and fast control of your ETHOS mining farms

== Description ==

Ethos-dashboard allows you to configure and create all the file to remote your mining Rig
1) Create your Wallet
2) Create your Group of Rig
3) Create each RIG
4) Put Each unique url for each RIG
5) Now you can manage all your rig in one clic, change wallet, RIG Group and more

== Installation ==

The installation of the plugin can be done in two ways:

* Method 1
1. Download the zip file from the WordPress website
2. Send the "fresh" folder to the `/ wp-content / plugins /` directory
3. Activate the plugin in the "Extensions" menu of WordPress

* Method 2
1. Search the plugin from the menu "Extensions" of WordPress
2. Start the installation of the plugin
3. Activate the plugin in the "Extensions" menu of WordPress

== Frequently Asked Questions ==

How it work ?

Ethos dashboard generates an unique link per RIG. You just have to copy this link in remote.conf. Then you can change everything, money, wallet, fanspeed, ... You managed all your mining farm very easily in WordPress !
